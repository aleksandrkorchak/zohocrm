# ZOHO CRM
Test project. Create a deal and assign a task to it 
<pre>
Use Open Server 5.2.2:
    WebServer: Apache-PHP-7.2
    Interpreter: PHP 7.2
    Database: MySQL 5.7

1) Download project from github
2) Run in terminal: composer update
3) Create database: zohocrm
4) Run:
      php artisan migrate
      php artisan db:seed
</pre>
